# Django Contact Form Project

## Project Info
Contact form using Python, Django and HTML.
User fills form and details are sent to company email automatically.

## Form Fields
- Name
- Phone Number
- Email
- Designation

## Features
- Form with all employee details
- Auto email on submit
- Success page after submission

## Tech Used
- Python
- Django
- HTML

## How to Run
1. pip install django
2. python manage.py migrate
3. python manage.py runserver
4. Open http://127.0.0.1:8000/
